# Dead Sector: Outbreak

Game bắn zombie sinh tồn góc nhìn thứ nhất dành cho desktop và mobile.

## Điều khiển

- Desktop: WASD, chuột, Shift, R, G, H và phím 1–2.
- Mobile: joystick trái, kéo bên phải để ngắm và các nút hành động cảm ứng.

Site được build bằng Vite và phát hành tự động qua GitHub Pages.
